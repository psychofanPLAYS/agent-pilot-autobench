# Support

## Start Here

1. Run `agent-autobench doctor --json-out` to see which local paths are missing.
2. Read [Start for Normal People](docs/START-FOR-NORMAL-PEOPLE.md).
3. Check the [command board](docs/COMMAND-BOARD.md) and existing
   [GitHub Discussions](https://github.com/psychofanPLAYS/agent-pilot-autobench/discussions).

## Ask a Question

Use GitHub Discussions for setup help, benchmark interpretation, and usage questions. Include
your operating system, Python version, llama.cpp build, command, and the smallest relevant
error excerpt. Never attach model files, credentials, private prompts, or full local receipts.

## Report a Bug or Request a Feature

Use the repository's issue forms. A small reproduction and sanitized receipt path are more
useful than a screenshot of a long terminal window.

Report security problems privately as described in [SECURITY.md](SECURITY.md).
